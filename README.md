# 🌙 Asma Ul Husna Website

An interactive web application showcasing the 99 Names of Allah with meanings, descriptions, favorites, audio, search functionality, and dark mode.

## ✨ Features

- 📖 99 Names of Allah
- 🔍 Search functionality
- ❤️ Favorite names system
- 🔊 Audio/TTS support
- 🌙 Dark & Light Mode
- 📱 Responsive design

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
